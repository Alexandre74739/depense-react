# Dépense V1
Cette application permet aux utilisateurs de suivre leurs dépenses dans
en ajoutant ou retirant des montants en fonction d'une date. L'objectif est de visualiser rapidement les dépenses.

## Fonctionnalités :

- Ajouter une date et une dépense
- Retirer une dépense
- Afficher le total
- Interface responsive
