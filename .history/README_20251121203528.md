# Dépense
Cette application permet aux utilisateurs de suivre leurs dépenses dans le mois
en ajoutant ou retirant des montants. L'objectif est de visualiser 
rapidement les dépenses du mois.

Fonctionnalités :

- Ajouter une dépense
- Retirer une dépense
- Afficher le total
- Interface moderne et responsive
